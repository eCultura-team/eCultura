# **Passos para instalar a aplicação eCultura** <h1>

### **Pre requisitos:** <h3>
  - Yarn ou NPM;
  - Node.js;
  - EXPO-CLI

### **Para instalar basta executar o seguinte comando na sua máquina:** <h3>
  - Yarn install ou NPM install

### **Agora é só rodar a aplicação!** <h3>
   - Yarn start ou NPM start
